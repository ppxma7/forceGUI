# forceGUI
A MATLAB App used for EMG force trace analysis

# Prerequisites:
Currently tested on:

MATLAB Version: 25.2.0.3150157 (R2025b) Update 4    
MATLAB License Number: 646274    
Operating System: Microsoft Windows 11 Home Version 10.0 (Build 26200)    

You will also need these MATLAB toolboxes. These can be downloaded by clicking on the 'Add-Ons' button on the top bar in MATLAB 2025.
- Signal Processing Toolbox
- Image Processing Toolbox
- Statistics and Machine Learning Toolbox
